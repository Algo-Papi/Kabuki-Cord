# Semi-auto Mode V2 Sprite Prompt

CLI fallback image generation used `gpt-image-2` at high quality. The accepted default scanner V2 atlas controlled the Kabuki actor identity, mask markings, costume, proportions, rendering, palette, camera, and scale. The original Semi-auto sheet supplied the broad action story only, while its clearest inspection frame supplied the bronze theatrical prop design.

The final brief requested exactly eight cells in a 4x2 atlas. It locked one inner-left holster and one prop throughout this restrained sequence: guarded stance; open the fixed robe fold; draw the prop downward with the trigger finger indexed; settle into compact low-ready; perform one chamber/status check with a single amber glint; lower safely; return the same prop to the same holster; close the robe and recover.

The prompt required identical barrel length, grip, bronze finish, muzzle, trigger guard, proportions, hand contacts, and holster position wherever visible. It prohibited firing, muzzle flash, bullets, cartridges, targets, impacts, smoke, recoil, aiming at the viewer, duplicate props or limbs, unrelated approval gestures, random glyphs, text, logos, watermarks, and any variation in the flat `#00ff00` chroma-key background.

The CLI source is `output/imagegen/semi-auto-v2-atlas-chromakey.png`, copied into the project as `design_assets/semi-auto-v2-atlas-chromakey.png`. The alpha-extracted source is `design_assets/semi-auto-v2-atlas-transparent.png`.
